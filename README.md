# Paldi-Front

**Paldi-Front** es la interfaz de usuario de **Paldi**, una aplicación diseñada para la gestión y control de cortinas, persianas, enrollables y demas para la empresa PALDI. Esta herramienta permite a los usuarios, servidores manejar de manera eficiente y centralizada las ventas de la empresa cotizaciones y generacion de reportes.

## Tecnologías Utilizadas

- **JavaScript**: Lenguaje de programación principal para la lógica de la aplicación.
- **HTML**: Lenguaje de marcado para estructurar el contenido web.
- **CSS**: Lenguaje de estilos utilizado para el diseño y la presentación visual de la aplicación.
- **Webpack**: Herramienta de empaquetado de módulos para aplicaciones JavaScript modernas.
- **Docker**: Plataforma para desarrollar, enviar y ejecutar aplicaciones en contenedores, facilitando la implementación y escalabilidad.

## Funcionalidades Clave

- **Control de Dispositivos**: Permite a los usuarios operar cortinas, persianas y otros sistemas similares de forma remota y programada.
- **Programación de Escenarios**: Facilita la creación de escenarios personalizados para automatizar la apertura y cierre de dispositivos según horarios o condiciones específicas.
- **Interfaz Intuitiva**: Ofrece un diseño limpio y fácil de usar, mejorando la experiencia del usuario en la gestión de sus dispositivos.

## Instalación y Configuración

Sigue estos pasos para configurar y ejecutar el proyecto en tu entorno local:

0. **Contacta con crisistecno para las configuraciones adicionales**:

1. **Clona el repositorio**:

   ```bash
   git clone https://github.com/CrisisTecno/Paldi-Front.git
   ```

2. **Navega al directorio del proyecto**:

   ```bash
   cd Paldi-Front
   ```

3. **Instala las dependencias**:

   ```bash
   npm install
   ```

4. **Configura las variables de entorno**:

   - Crea un archivo `.env` en la raíz del proyecto con las variables necesarias para la configuración.

5. **Inicia la aplicación**:

   ```bash
   npm start
   ```
